# Ferramenta de Busca de Fluxos Power Platform

## Como Usar

```bash
npm install
npm start
```

Abra **http://localhost:3000**

### Login
1. Clique em "Entrar com Microsoft"
2. Acesse **microsoft.com/devicelogin**
3. Digite o código exibido
4. Faça login com seu usuário

### Pesquisar
1. Digite o **nome do ambiente** (ex: `default-xxxxx`)
2. Use a barra de busca para filtrar por nome
3. Filtre por status (Ativado/Desativado)

### Ver Detalhes
- Clique em "Detalhes" para ver a definição JSON do fluxo
